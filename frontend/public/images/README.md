## How to Export from Figma

1. Open your Figma design file
2. Select the element/component you want to export
3. In the right sidebar, click the "Export" section
4. Choose PNG format
5. Set appropriate resolution (2x for retina displays)
6. Export and save to this directory with the exact filename listed above

## Note

The Figma API URLs require authentication and cannot be used directly in production.
All images must be exported manually and saved locally.
